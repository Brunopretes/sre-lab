# Laboratório SRE – Deploy Automatizado Simples

## Objetivo
Este laboratório simula um cenário real de um SRE Júnior responsável por criar um processo simples e confiável de deploy para uma aplicação web, garantindo testes antes da publicação, monitoramento básico e possibilidade de rollback manual.

---

## Tecnologias utilizadas
- Linux (Ubuntu / WSL)
- Python 3.11
- Flask
- Docker
- Pytest
- Bash
